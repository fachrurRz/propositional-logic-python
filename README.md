# Project Description
This project was made to fulfill computational final project. 

#### Feature
1. Parse String to Propositional Logic Tree
2. Get Formula and Subformulas
3. Evaluate logic assessment
4. Convert to CNF
5. Resolution
6. Entailment

### Propositional Logic Tree
           &
         /   \  
         P    ~
              |
              =>
            /   \
          Q       R
